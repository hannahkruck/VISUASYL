VISUASYL can be called up via a link without installation or executed locally. 


1. Run VISUASYL without installation: https://share.streamlit.io/hannahkruck/visuasyl/app.py
   Please use chrome or firefox browser for best performance.


2. If VISUASYL is to be run locally, the following steps are required:
	- Download GitHub VISUASYL Repository: https://github.com/hannahkruck/VISUASYL
	- Install all packages from requirements_base.txt and requirements_local.txt via command line: 
		pip install [Package that should installed]

	- To run VISUASYL you have to run the file --> app.py
	- Run VISUASYL with the command via command line: streamlit run [path/app.py]
